## Overview  

CyberSentinel is a cloud-native cybersecurity platform developed as a Final Year Engineering Project (2026).  

The system is designed to address modern cybersecurity challenges by combining infrastructure automation with an extensible architecture capable of integrating artificial intelligence-based threat detection mechanisms. It emphasizes scalability, reproducibility, and secure deployment practices aligned with DevSecOps principles.

---

## Problem Statement  

Traditional cybersecurity systems rely heavily on static rules and predefined signatures, which makes them ineffective against evolving and sophisticated threats. These systems often lack scalability and adaptability in dynamic environments.

CyberSentinel addresses these limitations by providing a modular and infrastructure-first approach that supports future integration of intelligent threat detection systems.

---

## Key Features  

- Infrastructure as Code using Terraform  
- Modular and extensible architecture  
- Cloud-ready deployment design  
- Security-focused system architecture  
- Support for future AI-based threat detection  
- Separation of infrastructure, logic, and testing layers

## System Architecture  

The project follows a layered architecture:

- Infrastructure Layer: Managed using Terraform to provision and maintain cloud resources  
- Application Layer (Planned): AI/ML-based threat detection modules  
- Automation Layer: Scripts for deployment and integration workflows  
- Testing Layer: Framework for validation and system testing
